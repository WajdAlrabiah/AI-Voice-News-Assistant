# AI Voice News Assistant

A Python-based **voice-enabled virtual assistant** that performs four main NLP-related tasks:
- Translates text from English to Arabic 
- Retrieves top headline news using the **News API** 
- Fetches real-time flight details via the **Aviation Stack API**
- Reads news headlines aloud using **Text-to-Speech (gTTS)**
